### 第二步：解決 Frontend (前端) 下載失敗問題

你個 Log 寫住 `WinError 2`，係因為程式想行 `git` 嚟下載個網頁介面，但佢搵唔到你個 Portable Git。

請喺 PowerShell 跟住呢幾句指令打（幫個程式搵返個 Git）：

```powershell
# 1. 暫時話俾電腦聽你個 Git 喺邊 (假設你 Git 喺 E:\git 資料夾入面)
$env:Path += ";E:\git"

# 2. 嘗試手動下載前端資料夾 (如果自動下載唔到)
git submodule update --init --recursive

# 3. 再次啟動程式
uv run run_server.py
```

### 喵喵小提醒：
*   **如果順利啟動**：你會見到 `Initializing Live2D: 新女兒`，然後見到 `Running on http://127.0.0.1:12393`。
*   **如果網頁開咗之後見唔到嘢 (Not Found)**：代表前端仲係未下載成功。到時話我知，我教你點樣手動用 `E:\git\git-cmd.exe` 夾硬 clone 佢落嚟。

試吓改咗個 `conf.yaml` 先，應該最少唔會再噴嗰大串紅色 Error 喵！🐾
