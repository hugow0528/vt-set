呢個係一個好好嘅問題！好多初學者都會問：「既然我都有個 App 介面咯，點解仲要搞咁多黑底白字嘅指令（Git, FFmpeg, uv）？」

我用一個簡單嘅比喻解釋畀你聽：

### 1. 點解一定要裝呢啲？（DVD 機比喻）

想像你要睇一套戲：
*   **Electron App（你見到嘅桌寵）：** 係嗰部「電視機」，負責顯示畫面。
*   **Git / Python / uv / Backend：** 係一部「DVD 機」同埋入面嘅「光碟」。
*   **FFmpeg：** 係連接電視同 DVD 機嘅「聲畫線」。

**如果你淨係開電視（App），而唔裝 DVD 機（Backend），電視只會係黑屏，因為冇訊號輸入。** 呢個 Project 嘅「大腦」係用 Python 寫嘅，所以你部電腦一定要有工具去行呢啲 Python 程式。

*   **Git：** 用嚟喺雲端「攞」個程式返嚟，同埋以後更新個程式。
*   **FFmpeg：** AI 聽你講嘢同埋佢講嘢都要處理音頻，冇 FFmpeg 佢就會變成「聾啞人士」。
*   **uv：** 佢係依家最快嘅 Python 管理員，負責幫你自動下載所有 AI 運行需要嘅零件。

---

### 2. 點樣安裝 Git 同相關工具（詳細步驟）

我哋唔使一個個網頁去搵，我哋用 Windows 內置嘅 **winget** 幫你一次過搞掂。

#### 第一步：打開「管理員版」PowerShell
1.  喺 Windows 掣（左下角）撳右鍵。
2.  揀 **「終端機 (管理員)」** 或者 **「Windows PowerShell (管理員)」**。

#### 第二步：安裝 Git 同 FFmpeg
喺 PowerShell 入面 Copy 下面呢行，撳 Enter：
```powershell
winget install Git.Git ffmpeg
```
*   **注意：** 如果佢問你同唔同意條款，打 `y` 撳 Enter 就得。
*   **點檢查裝好未？** 打 `git --version` 同 `ffmpeg -version`，有版本號碼出嚟就係成功。

#### 第三步：安裝 uv (Python 管理器)
依然喺 PowerShell，Copy 下面呢行長指令：
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
*   裝完之後，**一定要熄咗個 PowerShell 視窗，再開返一個新嘅**（或者重啟電腦），咁樣電腦先會認得 `uv` 呢個指令。

---

### 3. 下載並設定「大腦」(Backend)

依家我哋要用 Git 嚟攞程式碼喇：

1.  喺 PowerShell 入面，先入去你想放程式嘅地方（例如 D 盤）：
    ```powershell
    d:
    ```
2.  **用 Git 下載程式：**
    ```powershell
    git clone https://github.com/Open-LLM-VTuber/Open-LLM-VTuber.git
    ```
3.  **入去嗰個資料夾：**
    ```powershell
    cd Open-LLM-VTuber
    ```
4.  **用 uv 幫你自動裝埋 Python：**
    （呢步好神奇，佢會自動幫你下載埋啱用嘅 Python 版本，你唔使自己去官網搵）
    ```powershell
    uv sync
    ```
    *呢步會行好耐，因為佢下載緊好多 AI 零件。見到完成（Done）就得。*

---

### 4. 設定與啟動（最後衝刺）

1.  **準備 Config 檔：**
    你可以喺資料夾入面手動 Copy，或者喺 PowerShell 打：
    ```powershell
    copy config_templates\conf.ZH.default.yaml conf.yaml
    ```
2.  **放入你個 Live2D：**
    將你個 Live2D 模型資料夾，擺入去呢個路徑：
    `D:\Open-LLM-VTuber\src\open_llm_vtuber\resources\live2d\`
3.  **啟動大腦：**
    ```powershell
    uv run run_server.py
    ```

---

### 5. 總結：你以後每日點樣「開工」？

當你裝好晒上面啲嘢之後，你以後每日只需要做呢兩件事：

1.  **開大腦：** 入去 `Open-LLM-VTuber` 資料夾，開 PowerShell 打 `uv run run_server.py`。（唔好熄咗個視窗，縮細佢就得）
2.  **開 App：** 打開你下載嗰個 `open-llm-vtuber-web` (Electron) App。

**點解要咁煩？**
因為呢個係一個「開源科研項目」，唔係一個商品化嘅軟件。你要用呢啲工具嚟將你部電腦變成一個「AI 工作站」。
